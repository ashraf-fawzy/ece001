features of programming languages :

1) Low level programming languages

Advantages Disadvantages
Programs developed using low level languages are fast and memory efficient. Programs developed using low level languages are machine dependent and are not portable.
Programmers can utilize processor and memory in better way using a low level language. It is difficult to develop, debug and maintain.
Low level languages provide direct manipulation of computer registers and storage. Low level programming usually results in poor programming productivity.
It can directly communicate with hardware devices. Programmer must have additional knowledge of the computer architecture of particular machine, for programming in low level language.

2) High level programming languages

Advantages Disadvantages
High level languages are programmer friendly. They are easy to write Easy to learn, debug and maintain. It takes additional translation times to translate the source to machine code.
It provide higher level of abstraction from machine languages. High level programs are comparatively slower than low level programs.
It is machine independent language. Compared to low level programs, they are generally less memory efficient.
Less error prone, easy to find and debug errors. Cannot communicate directly with the hardware.

3) Middle Level programming language

Hence, we can say that the programming languages which have features of Low Level as well as High Level programming languages known as "Middle Level" programming language. C programming languages is the best example of Low Level Programming languages as it has features of low level and high level programming languages both.
